# SubnetCalculator

SubnetCalculator is a lightweight and web-based tool to calculate different networks and subnets created by Justin Verstijnen. This tool has some templates which you can use or you can define your own IP ranges.

## Main features

The main features of this tool are:

- Calculate network information:
  - Network address
  - Broadcast address
  - First and last usable IP
  - Total hosts
  - Usable hosts
- Supports both subnet mask and CIDR notation input
- Clear, formatted output
- Export results to different formats (e.g. JSON, CSV)

## Hosting
